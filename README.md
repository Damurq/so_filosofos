# Cena de los Filosofos
Solucion al problema cena de los filosofosos.

Debes tener instalado pytohn 3 o mayor

Entra a la carapeta raiz y ejecuta

`python filosofos.py` or `python3 filosofos.py`

##  Las varaibles a configurar:
Número de filosofos
``N``

Representa el número maximo de comidas por filosofo
``MAXIMUM_NUMBER_MEALS``

Tiempo de espera entre las comidas y pensamientos 
 --Este en una lista de dos numeros que representa un intervalo que se utilizará para generar una
 --cantidad de segundos aleatorios (en caso de ser iguales el número de segundos es constante)
``STANDBY_TIME = [1,1]``

Número de tenedores
``TENEDR``
